This MATLAB FileExchange entry contains files related to the "Motion Estimation" video of the AUVSI Foundation: Computer Vision Training.

examples - Files used in the video
[~] motionPerception.m : Example MATLAB code file to show the effect of motion perception.
[~] estMotionOF_start.m : Example starter MATLAB code file to estimate motion of a robot boat.
[~] estMotionOF.m : Example MATLAB code file to estimate motion of a robot boat.
[~] ofRunTime.m : Example MATLAB code file to see the computation time of optical flow methods.
[~] buoyMotion.avi : Example video of a robot boat navigating a field of red and green buoys.

exercises - Files for exercise and practice tasks 
[~] auvsi_ipcv_exercises_07_note.pdf : Note introducing the exercise tasks and presenting respective solutions.
[~] estMotionHand_start.m : Example starter MATLAB code file for the exercise "Hand Motion".
[~] estMotionHand.m : Solution MATLAB code file for the exercise "Hand Motion".
[~] estMotionBall_start.m : Example starter MATLAB code file for the exercise "Motion Estimation of a Ball".
[~] estMotionBall.m : Solution MATLAB code file for the exercise "Motion Estimation of a Ball".
[~] thumb.avi : Example video of a moving hand for the exercise "Hand Motion".
[~] multiObject.avi : Example video of 2 balls moving towards left and right for the exercise "Motion Estimation of a Ball".
